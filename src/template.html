Tip Calculator

This project will allow you to calculate the Tip for the waitress in a restaurant. It allows the end user to enter: - The total bill amount; - The Service level: Outstanding-30%, Good-20%, OKish-15%, Bad - 10%, Never coming back - 0%; - The number of persons that are splitting the bill. In the end, it returns the Tip value for each person based on next calculation: (TotalBillAmount * ServiceLevel)/NumberOfPersons.
